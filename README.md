# 🛒 Marketplace App

A modern Marketplace web application built with **React + TypeScript**.

---

## 🚀 Features (Planned / In Progress)

- 🔐 User Authentication (Register / Login)
- 📝 Create & Browse Listings
- 📂 Categories & Search
- ❤️ Favorites
- 💬 Chat simulation with sellers
- 🌙 Dark / Light Mode
- 📱 Fully Responsive Design

---

## 🧠 Tech Stack

- React 18 + TypeScript
- Vite
- React Router DOM
- Context API + LocalStorage
- Tailwind CSS
- Lucide React (Icons)
